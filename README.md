# BRBMDL
Belief Rule-Based Model with Deep Learning
基於信念規則的多因子深度學習模型以預測股票價格

近幾年的股價預測研究，普遍使用的是黑箱模型，像是LSTM, CNN, Transformer等，但在股票市場這樣高風險的環境，要將預測模型應用到實務中，必須考慮到合規性、風險等等，因此，模型的可解釋性就至關重要。

本研究提出一種基於信念規則的多因子深度學習模型，用於預測股票價格。這個模型結合了信念規則證據推理和深度學習技術，並且納入了多種影響股價的因子（包括歷史股價、技術指標和情緒極性）。
旨在克服普遍股價預測模型可解釋性不足的問題，同時保持模型的準確性，為投資決策提供更可靠的支持。
